# 支援向量機 Support Vector Machine

## 核心直覺
支援向量機尋找能把類別分開、且間隔最大的決策邊界。距離邊界最近的樣本稱為支援向量。

## 最大間隔
SVM 希望分類正確，同時讓邊界與最近樣本之間的距離最大。

$$
\min \frac{1}{2}\|w\|^2
$$

## Soft Margin
真實資料常不可完全線性分開，因此用參數 `C` 控制容錯程度。

- `C` 大：較少容忍錯誤，可能過擬合。
- `C` 小：容忍更多錯誤，邊界較平滑。

## Kernel Trick
核技巧可在不明確建立高維特徵的情況下，處理非線性決策邊界。

常見 kernel：
- Linear
- Polynomial
- RBF
- Sigmoid

## 優點
- 在中小型資料集表現穩定。
- 高維資料也能有效運作。
- RBF kernel 可處理複雜邊界。

## 限制
- 大型資料訓練成本高。
- 參數與特徵縮放很重要。
- 機率輸出不是原生結果。

## 常見應用
- 文字分類
- 影像特徵分類
- 生物資訊分類
- 異常偵測

## 小練習
先標準化特徵，再比較 linear kernel 與 RBF kernel 的決策邊界。
