# 神經網路 Neural Network

## 核心直覺
神經網路由多層神經元組成，每層把輸入做線性轉換，再套用非線性激活函數。多層堆疊後，模型能學到複雜模式。

## 基本單元
$$
a=\phi(Wx+b)
$$

其中 `phi` 可以是 ReLU、sigmoid、tanh 等激活函數。

## 訓練流程
1. 前向傳播：計算預測。
2. 計算損失：比較預測與真實標籤。
3. 反向傳播：計算每個參數的梯度。
4. 優化器更新：例如 SGD、Adam。
5. 重複多個 epoch。

## 常見技巧
- 標準化輸入。
- 使用 ReLU 改善深層訓練。
- 用 dropout 或 weight decay 降低過擬合。
- 用 validation loss 做 early stopping。

## 優點
- 表達能力強。
- 適合影像、語音、文字等複雜資料。
- 可透過架構設計納入領域知識。

## 限制
- 需要較多資料與計算資源。
- 可解釋性較低。
- 參數、架構與訓練流程都需要仔細調整。

## 常見應用
- 影像辨識
- 語音辨識
- 自然語言處理
- 時序預測

## 小練習
觀察 learning curve：如果訓練分數很高但驗證分數低，代表可能過擬合，可以加入正則化、資料增強或早停。
